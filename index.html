Instructions

  • Colour: Select which colour the squares will be painted when you hover the mouse over them
  • Brush (On/Off): Will let you know the current status of your brush
  • Background: When clicked all the squares will be changed to the selected colour
  • New Grid: When clicked you will be prompted to choose the number of squares in each row for a newly created grid
